## Overview
This is a custom-built macropad designed for enhanced PC productivity and gaming. It features 6 mechanical keys that can be programmed for macros, shortcuts, and keybinds.

An OLED display is used to show real-time information such as active profiles, macro labels, or system/game status.

## Features
- 6 mechanical mechanical keys
- Programmable macros and shortcuts
- OLED display for live status
- USB HID device
- Compact external controller

## Tech Stack
- Microcontroller (Arduino / RP2040 / ESP32)
- Mechanical switches
- OLED display
- Arduino HID / QMK firmware
