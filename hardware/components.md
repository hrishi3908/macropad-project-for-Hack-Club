## Components List

### Main Components
- Microcontroller (Arduino Pro Micro / RP2040 / ESP32)
- 6x Mechanical keyboard switches
- Keycaps (compatible with switches)
- 0.96 inch OLED display (I2C, SSD1306)

### Supporting Components
- Jumper wires
- USB cable (for power + data)
- Breadboard or custom PCB (future)
- Diodes (only if key matrix is used)

### Notes
Component selection will be finalized based on availability, firmware compatibility, and ease of HID implementation.
